## 参加方法
手順は以下の通りです．

1. @Homeリーグ実行委員が用意する仮登録フォームに申請する（本登録で必要なチームコードが発行されます）
2. ロボカップ事務局へ本登録を行う

詳細については，後日アナウンスいたしますので，しばらくお待ちください．
